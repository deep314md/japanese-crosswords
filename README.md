# japanese-crosswords

This program is looking for the solution for Japanese crosswords. 
Initially the program was written in the PHP language 
 - newjapan.php
  
Then transcribed into Python, where the function was divided into several files.
 
 - japanDo.py : here is the core of the program, which coordinates with all actions
  
 - japanFunc.py : the basic functions that process information
 
 - japanFiltre.py : filtering functions of possible variants in search of the solution
 
 - varParams.py : here are the constants used and the parameters for our program
